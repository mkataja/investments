# SSGA / State Street holdings XLSX

**HTTPS** daily **XLSX** on **`ssga.com`**. Distribution source key **`ssga_holdings_xlsx`**. Parser **`api/src/distributions/parseSsgaHoldingsXlsx.ts`**.

Cash / cash-equivalent rows (e.g. **Asset Class**, **Security Type**, or **Sector Classification**) → **`sectors.cash`**, omitted from **`countries`**. See [Provider holdings overview](../data-sources.md#provider-holdings-overview).
