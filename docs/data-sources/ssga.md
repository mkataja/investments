# SSGA / State Street holdings XLSX

HTTPS daily XLSX on `ssga.com`. Source key `ssga_holdings_xlsx`. Parser `api/src/distributions/parseSsgaHoldingsXlsx.ts`.

Cash rows (e.g. Asset Class, Security Type, Sector Classification) → `sectors.cash`, omitted from `countries`. [Provider holdings overview](../data-sources.md#provider-holdings-overview).
