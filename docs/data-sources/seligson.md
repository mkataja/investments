# Seligson mutual funds

## Fund name and NAV

The API **fetches Seligson HTML** to resolve **`name`** when inserting a new **`seligson_funds`** row ( **`fetchSeligsonFundName`** ). Failures are surfaced as HTTP errors with a message body. Exact routes and status codes are defined in **`api`**—read there before changing.

**FundValues NAV (`FundValues_FI.html`)** uses shorter link text than FundViewer in some rows; **`fundValuesRowMatchesDbName`** includes **aliases** (e.g. table **`Global Brands`** ↔ DB name containing **Top 25 Brands**) in **`FUND_VALUES_TABLE_LABEL_ALIASES`** in **`api/src/distributions/seligsonFundValues.ts`**.

## Holdings distribution (HTML scrape)

**HTML scrape** of Seligson FundViewer **holdings listing (view=10)**; **`fid`** in the URL must match **`seligson_funds`** in the DB. Users register funds via **`/instruments/new`** (FID + **Seligson-type** broker); **`seligson_funds`** rows are **find-or-created** by the API, not edited via a separate admin app. **Geo** from each row’s Finnish **Maa** → ISO via **`resolveRegionKeyToIso`**. **Sectors:** line weights use Yahoo **`quoteSummary` `assetProfile.sector`** (fallback **`assetProfile.industry`** when sector is empty) mapped by **`mapSectorLabelToCanonicalId`** when resolution succeeds (**`yahooFinance.search`** via **`buildYahooSearchQueriesForSeligson`** (**`expandCoTheStyleSearchQueries`** rewrites **`cos inc/the`**, **`co/the`**, **`corp/the`**, **`/the`**, **`inc/md`**, trailing **`llc`**, prepends **`carrefour`** for truncated Carrefour legal names, **`ABX.TO`**/**`Barrick Mining`** and **`AEM`**/**`agnico eagle`** for Canadian miners, plus aliases like **`dsv`**, **`lvmh`**, **`sabanci holding`**, **`concha y toro`**; **`FI`:** prepends **`{token} pankki`** when **`Oyj`** strips to one word, and **`outotec`‑less** names when **`outotec`** appears; **`JP`:** prepends search without trailing **`kk`**; **`HK`:** prepends **`hong kong exchange and clearing`** and **`HKEX`** when the name is HKEX-style; **`DK`:** prepends **`NSIS-B.CO`** for **`novonesis`**; merges with **`Oyj`**‑stripped and original queries) then **`namesMatchSeligsonYahoo`** (**`novozymes`**→**`novonesis`**, **`barrick mining`**→**`barrick gold`**, truncated Carrefour **`cie`/`compagnie` `generale des etablissement(s)`**→**`carrefour`** in **`collapseSplitLegalAbbrevs`**; **`FR`:** prepends **`CA.PA`** for that Carrefour pattern): normalizes **`&`** vs **and**, **Spanish** standalone **y** as **and**, Nordic **`a/s`** as a placeholder (**not** `as`, which would be stripped as Turkish legal form), **`boerse`** as **`borse`** (Yahoo **ö**), **`Cie`** as **`compagnie`**, **`Hong Kong Exchanges`** as **`Hong Kong Exchange`**, **`hkex`**/**`nyk`** expanded to **`hong kong exchange and clearing`**/**`nippon yusen`** for Yahoo short names, strips **`/Japan`** / **`/Canada`** / **`/Delaware`** / trailing **` of`** / trailing **`-`** (truncated words), merges **`l`+`oreal`**, German **`ue`** spellings (**`muenchener`**, **`rueck`…**) to match NFKD-stripped umlauts, Turkish **ı/İ** to ASCII **`i`**, standalone **`Corp`** tokens expanded to **`corporation`**, **`Cos` `inc`** to **`Companies inc`**, **`Co` `the`** (from **co/the**) to **`Company the`**, trailing legal tokens including **Oyj** / **Pc** / **AS** / **KK** / **SGPS** / **SAB** / **CV** / **de** / **delaware** / **kabushiki** / **kaisha** / **holdings** / **reit** / **trust** / **group** / **cos** vs **Companies**, etc., substring match, **initials** for short tickers like **ANZ**, **two-letter** tickers as a word (e.g. **EQ**) with a denylist for ambiguous pairs, **significant-word** coverage for short names like **Link REIT** vs long trust titles; **quoteSummary** ISIN equal to row ISIN **accepts without name check**; **no** Yahoo listing-country vs Seligson **Maa** check; **search** and **OpenFIGI** symbol lists try **each** candidate until name/ISIN **and** a mappable sector/industry succeed (many APAC symbols have empty sector on one listing). If that fails and an **ISIN** appears in row markup, **OpenFIGI** + Yahoo symbol candidates as fallback); **first** resolvable candidate wins; otherwise **Toimiala** → **`SELIGSON_FINNISH_SECTOR_LABEL_MAP`** (unknown → **`sectors.other`**). Wrong ticker/name pairs are possible—**manual validation** of cached resolutions may be added later. Resolved lines are cached globally in **`seligson_holdings_resolution_cache`** (normalized Seligson company name + ISO country code, **`ZZ`** when **Maa** does not map; Yahoo **`yahoo_symbol`** and **`yahoo_company_name`** when `source = yahoo`; no TTL). Cash / money-market lines (e.g. **Käteinen**, or **Maa**/**Toimiala** dash-only) → **`sectors.cash`** and **omitted** from **`countries`**. Raw HTML is **`seligson_distribution_cache.holdings_html`**. **`YAHOO_MIN_INTERVAL_MS`** spaces **distinct** unresolved lookup keys and **OpenFIGI** fetches when the ISIN fallback runs (not between Yahoo candidate tries for one key).

Main modules: **`api/src/distributions/seligson.ts`**, **`api/src/distributions/seligsonHoldingsResolve.ts`**.
