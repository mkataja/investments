# DWS Xtrackers constituent XLSX

HTTPS constituent XLSX on `dws.com` (e.g. `etf.dws.com/.../excel/product/constituent/...`). Source key `xtrackers_holdings_xlsx`. Parser `api/src/distributions/parseXtrackersHoldingsXlsx.ts`.

Cash rows (Type of Security) → `sectors.cash`, omitted from `countries`. [Provider holdings overview](../data-sources.md#provider-holdings-overview).
