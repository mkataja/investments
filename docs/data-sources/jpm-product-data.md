# J.P. Morgan product-data URL (sector breakdown JSON)

**`instruments.provider_breakdown_data_url`** — HTTPS **`am.jpmorgan.com/.../FundsMarketingHandler/product-data`** (JSON **`emeaSectorBreakdown`**). **`validateProviderBreakdownDataUrl`** in **`@investments/db`**; only when **`holdings_distribution_url`** is a JPM daily ETF XLSX URL (see [jpm-holdings.md](./jpm-holdings.md)). Geographic weights stay from the XLSX; sector weights from **`parseJpmProductDataSectorBreakdown`** (skip **`Total`** aggregate rows; portfolio % from each row’s **`value`** only — **`secondaryValue`** is benchmark/comparison, not used); XLSX cash is merged into **`sectors.cash`**. **`version`** (and other query params) are part of the stored URL—copy from the fund page network tab if requests fail.
