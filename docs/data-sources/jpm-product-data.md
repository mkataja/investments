# J.P. Morgan product-data URL (sector breakdown JSON)

`instruments.provider_breakdown_data_url` — HTTPS `am.jpmorgan.com/.../FundsMarketingHandler/product-data` (JSON `emeaSectorBreakdown`). `validateProviderBreakdownDataUrl` in `@investments/lib`; only valid when `holdings_distribution_url` is a JPM daily ETF XLSX URL ([jpm-holdings.md](./jpm-holdings.md)). Geography from XLSX; sectors from `parseJpmProductDataSectorBreakdown` (skip `Total` rows; portfolio % from `value` only — `secondaryValue` is benchmark, not used); XLSX cash merged into `sectors.cash`. Copy full URL including `version` from the fund page network tab if requests fail.
